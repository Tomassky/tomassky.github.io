Kali Linux笔记(三十六)：Web渗透实验环境及工具

发表于 2019-12-27 | 分类于 Kali
字数统计: 267 字 | 阅读时长 ≈ 1 分钟

1. 实验环境

  • Metasploitable（访问浏览器Metasploitable的网页）
    • Dvwa（不同级别Low → Medium → High）

2. 侦察

  • Httrack
    • 对目标网站的克隆复制
    • 减少与目标系统交互
  • Httrack
    • Enter Project name
    • Base path（要ms拷贝的目录，大目录）
    • Enter URLs（起始路劲）
    • Action（一般选Mirror Web Site with Wizard）
    • Proxy
    • wildcard（定义通配符）
    • Additional options（其余选项）
    • Ready to launch the mirror
可选择的Action：
Mirror Web Site：镜像网页站点
Mirror Web Site with Wizard：网页向导
Just Get file Indicated：获得某种类型的文件
Mirror ALL links in URLs（Multiple Mirror）：镜像所有连接
Test Links In URLs（Bookmark Test）：只测试连接
▲ 代理可以通过 Hide My Ass来进行掩藏（可能需要代理Proxy List）

3. 扫描工具

  • Nikto（纯主动扫描工具）
  • Vega
  • Skipfish
  • W3af
  • Arachni
  • Owasp-zap
Web扫描工具模式
1.代理截断模式：代理扫描截断
2.主动扫描模式：主动扫描，注入代码
▲ 自动扫描相对于手动扫描，更容易找到隐藏的页面，能提高很大的效率，但容易触发警报
