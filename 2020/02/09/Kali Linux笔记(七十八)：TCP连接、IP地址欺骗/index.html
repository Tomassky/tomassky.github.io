2. IP地址欺骗

  • 经常用于DoS攻击
  • 根据IP头地址寻址
    • 伪造IP源地址
  • 边界路由过滤器
    • 入站、出站
  • 受害者可能是源、目的地址
  • 绕过基于地址的验证
  • 压力测试模拟多用户
  • 上层协议（TCP序列号）▲ TCP链路劫持
  • ▲ ISP设置出站流量的策略，对伪造源地址出站流量的过滤（边界防火墙）
