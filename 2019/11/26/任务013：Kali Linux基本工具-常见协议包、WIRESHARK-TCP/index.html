发表于 2019-11-26 | 分类于 Kali

  1. 数据包的分层结构

    • ARP
    • ICMP
    • TCP-三次握手
    • UDP
    • DNS
    • HTTP
    • FTP
对于非官方端口解析：选中 → 右键 → Decode As → HTTP

  1. WIRESHARK-TCP

    • 数据流
      • http（明文传输）
      • Smtp（明文传输）
      • Pop3（明文传输）
      • Ssl（密文传输，公钥证书是以明文传输的形式）
完整的数据响应过程：选中 → 右键 → follow TCP/UDP stream
